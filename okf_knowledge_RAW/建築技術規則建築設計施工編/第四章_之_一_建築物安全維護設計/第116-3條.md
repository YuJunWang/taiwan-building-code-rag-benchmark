# 第116-3條

安全維護照明裝置照射之空間範圍，其地面照度基準不得小於下表規定：
┌───┬─────────────────┬────────┐
│      │空間種類                          │照度基準（lux） │
├───┼─────────────────┼────────┤
│（一）│停車空間（室內）                  │六十            │
├───┼─────────────────┼────────┤
│（二）│停車空間（室外）                  │三十            │
├───┼─────────────────┼────────┤
│（三）│車道                              │三十            │
├───┼─────────────────┼────────┤
│（四）│車道出入口                        │一百            │
├───┼─────────────────┼────────┤
│（五）│安全梯間                          │六十            │
├───┼─────────────────┼────────┤
│（六）│公共廁所                          │一百            │
├───┼─────────────────┼────────┤
│（七）│基地內通路                        │六十            │
├───┼─────────────────┼────────┤
│（八）│避難層出入口                      │一百            │
└───┴─────────────────┴────────┘
