# 第167-7條

建築物使用類組為 B-4  組者，其無障礙客房數量不得少於下表規定：
┌────────────────┬─────────────┐
│客房總數量（間）                │無障礙客房數量（間）      │
├────────────────┼─────────────┤
│十六至一百                      │一                        │
├────────────────┼─────────────┤
│一百零一至二百                  │二                        │
├────────────────┼─────────────┤
│二百零一至三百                  │三                        │
├────────────────┼─────────────┤
│三百零一至四百                  │四                        │
├────────────────┼─────────────┤
│四百零一至五百                  │五                        │
├────────────────┼─────────────┤
│五百零一至六百                  │六                        │
├────────────────┴─────────────┤
│超過六百間客房者，超過部分每增加一百間，應增加一間無障礙客房│
│；不足一百間，以一百間計。                                  │
└──────────────────────────────┘
