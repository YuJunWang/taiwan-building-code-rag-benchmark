# 第308條

建築物建築外殼節約能源之設計，應依據下表氣候分區辦理：
┌─────┬────────────────────────┐
│氣候分區  │行政區域                                        │
├─────┼────────────────────────┤
│北部氣候區│臺北市、新北市、宜蘭縣、基隆市、桃園縣、新竹縣、│
│          │新竹市、苗栗縣、福建省連江縣、金門縣            │
├─────┼────────────────────────┤
│中部氣候區│臺中市、彰化縣、南投縣、雲林縣、花蓮縣          │
├─────┼────────────────────────┤
│南部氣候區│嘉義縣、嘉義市、臺南市、澎湖縣、高雄市、屏東縣、│
│          │臺東縣                                          │
└─────┴────────────────────────┘
