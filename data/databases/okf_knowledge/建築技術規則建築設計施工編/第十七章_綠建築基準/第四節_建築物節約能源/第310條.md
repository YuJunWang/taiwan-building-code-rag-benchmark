# 第310條

住宿類建築物外殼不透光之外牆部分之平均熱傳透率應低於三點五瓦／（平方公尺．度），且其建築物外殼等價開窗率之計算值應低於下表之基準值。但符合本編第三百零八條之二規定者，不在此限。

│住宿類：    │  氣候分區  │建築物外殼等價開窗率基準值  │
│Ｈ類第一組  ├──────┼──────────────┤
│Ｈ類第二組  │北部氣候區  │百分之十三                  │
│            ├──────┼──────────────┤
│            │中部氣候區  │百分之十五                  │
│            ├──────┼──────────────┤
│            │南部氣候區  │百分之十八                  │
