# 第302條

建築基地之綠化，其綠化總固碳當量應大於二分之一最小綠化面積與下表固碳當量基準值之乘積：
┌──────────┬───────────────────┐
│使用分區或用地      │固碳當量基準值                        │
│                    │（公斤／（平方公尺‧年））            │
├──────────┼───────────────────┤
│學校用地、公園用地  │零點八三                              │
├──────────┼───────────────────┤
│商業區、工業區（不含│零點五零                              │
│科學園區）          │                                      │
├──────────┼───────────────────┤
│前二類以外之建築基地│零點六六                              │
│                    │                                      │
└──────────┴───────────────────┘
